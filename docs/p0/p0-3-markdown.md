## 1.标题

\# 一级标题

# 一级标题

\## 二级标题

## 二级标题

\### 三级标题

### 三级标题

直至 六级标题

## 2.列表

### 2.1.无序列表（无序号）

\- + 空格 + 内容

\- 无序列表项1

- 无序列表项1

\- 无序列表项2

- 无序列表项2

### 2.2.有序列表（有序号）

数字 + 空格 + 内容

1. 有序列表项1
2. 有序列表项2
3. ...

## 3.引用

\> 内容

> 引用

## 4.任务列表

**任务列表稍微复杂一点：\- + 空格 + [ ] + 空格**

\- [ ]

- [ ]

## 5.代码块

\``` 按哪种编程语言显示

内容

\```

```python
print("小白慢爬营")
```

对于行中文字进行标识，通常用两个 \`将文字包裹，如：\`文字\`。
